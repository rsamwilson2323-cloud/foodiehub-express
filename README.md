## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/rsamwilson2323-cloud/foodiehub-express.git

cd foodiehub-express
```

### 2️⃣ Install Dependencies

```bash
npm install
```

or

```bash
bun install
```

### 3️⃣ Start Development Server

```bash
npm run dev
```

or

```bash
bun run dev
```

Windows users can also launch:

```bash
run.bat
```
